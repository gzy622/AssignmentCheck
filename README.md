# 作业极速登记

一个极简、高效的作业登记与管理系统，专为教师设计，支持快速记录和统计学生作业提交情况。本项目采用单文件架构，资源完整本地化，支持离线使用，是您课堂管理的得力助手。

## 核心特性

- **多模式交互**：
  - **登记模式**：单击卡片即刻切换提交状态，适合快速点名登记。
  - **评分模式**：支持 A/B/C/D 评分或自定义分数，单击弹出评分窗口。
  - **长按快捷键**：无论在何种模式下，长按学生卡片即可快速进入评分界面。
- **沉浸式体验**：
  - **一键沉浸**：点击右下角展开按钮，隐藏所有导航栏，全屏展示学生网格，适合大屏演示或专注操作。
  - **姓名/学号切换**：支持仅显示学号或显示姓名缩写，兼顾隐私与识别效率。
- **多作业管理**：
  - 支持创建、重命名、删除多个作业任务。
  - 实时显示各作业的提交人数统计与进度条。
- **全方位统计分析**：
  - **多维度统计**：支持跨作业综合统计，生成班级完成率、排行榜。
  - **预警功能**：自动筛选完成率低于 50% 的“需要关注”学生名单。
  - **一键分享**：自动生成详尽的统计报告（含 Top 10、缺交名单等），支持一键复制。
- **数据安全与灵活**：
  - **备份导出**：一键导出 JSON 格式数据，支持多设备同步或本地存档。
  - **一键反选**：快速处理全班提交或个别补交场景。
- **移动端优化**：
  - 针对手机端设计的网格布局，支持触控振动反馈。
  - 智能键盘适配，自动调整弹窗位置，防止输入框被遮挡。

## 技术栈

- **前端框架**：Tailwind CSS (本地化，v3.x)
- **图标系统**：Font Awesome 6 Pro (本地化，v6.1.1)
- **核心逻辑**：Vanilla JavaScript (原生 JS，无外部依赖)
- **数据存储**：LocalStorage (本地离线存储)

## 本地化资源

为了保证在无网络环境（如封闭教室）下依然能稳定运行，所有资源已完整离线化：
- Font Awesome 图标库完全本地化至 `css/` 和 `webfonts/`。
- Tailwind CSS 样式表已本地化至 `css/tailwind.min.css`。

## 版本管理

### 版本规则

本项目遵循语义化版本：`主版本.次版本.修订号`。
- **主版本**：重大交互变更或架构方案调整。
- **次版本**：新增核心功能（如统计报告、沉浸模式等）。
- **修订号**：UI 细节优化、Bug 修复或性能推敲。

### 当前版本

目前的最新版本为：**v1.1.2**

### 如何更新版本号

1. 修改 `index.html` 中的 `APP_VERSION` 常量：
```javascript
const APP_VERSION = '1.1.2'; // 修改此处
```
2. 同时更新 `index.html` 中数据管理弹窗底部的显示：
```html
<span id="data-modal-version">1.1.2</span> <!-- 修改此处 -->
```

## 使用小技巧

1. **快速补交**：在数据管理中点击“一键反选”，然后再取消已交学生，即可快速记录迟交名单。
2. **长按大法**：即使在“登记模式”想偶尔打个分，也不需要切换模式，直接长按该学生即可。
3. **统计生成**：点击“统计”按钮后，可以选择多个作业一起看平均表现，系统会自动识别表现优异和落后的同学。
